2022年未來內容原型開發支持方案作業要點

切結書

茲保證本單位所提 （以下簡稱本計畫）及立切結書人（即申請人）符合下列事項：

  1. 承諾申請補助計畫之文件、資料均無虛偽不實、隱匿之情事，且符合「2022年未來內容原型開發支持方案作業要點」規定。
  2. 擔保本公司/單位無大陸地區人民、法人、團體、其他機構所直接/間接持有股份或出資總額超過30％，或受其實質控制之情事。
  3. 相同或類似之申請案計畫書未獲文化內容策進院、文化部及其附屬機關(構)、文化部單位預算下補（捐）助之相關單位（包含但不限於財團法人國家文化藝術基金會）及行政法人（包含但不限於及國家表演藝術中心）等單位經費支持。
  4. 本計畫內容未經發表，且未授權第三方使用。
  5. 承諾本計畫案如獲文化內容策進院經費支持，將以申請者名義執行。
  6. 本計畫案如獲文化內容策進院經費支持，承諾依該院核定之計畫書所載內容執行，且需符合「2022年未來內容原型開發支持方案作業要點」規定；計畫書有變更者，亦同。
  7. 本計畫案如獲文化內容策進院經費支持，承諾不會將獲支持資格或計畫書轉讓予他人。
  8. 本計畫案如獲文化內容策進院經費支持，擔保計畫書內容及依計畫書所辦理之各項工作，均無侵害他人權利或違反法律規定之情事。
  9. 本計畫案如獲文化內容策進院經費支持，承諾不會將支持經費用於添購軟硬體設備、裝修或修繕等費用，以及增加投資等相關事項上。如有違反，同時文化內容策進院得要求繳回已撥之支持經費。
  10. 本計畫案如獲文化內容策進院經費支持，承諾於計畫執行期間，將依文化內容策進院及文化部要求提供書面資料或出席會議說明，並依文化內容策進院之意見確實執行。
  11. 本計畫案如獲文化內容策進院經費支持，將自簽約日起至該院核撥最後一期支持金次日後一年期間內，應配合出席計畫相關宣傳活動。
  12. 本計畫案如獲文化內容策進院經費支持，承諾將依《預算法》第六十二條之一規定，不以置入性行銷進行政策宣導。如有政策宣導，應標示為「廣告」，並揭示文化內容策進院及文化部全銜。
  13. 本計畫之內部團隊成員與跨國合製案將依中國民國所得稅法及相關法令規定辦理所得申報扣繳及歸戶事宜，如有不實，願負一切法律責任。
  14. 本計畫案如獲文化內容策進院經費支持，承諾於所完成之開發成果，如有相關宣傳訊息或宣傳物等，將於適當處標示文化內容策進院LOGO。
  15. 本計畫案如獲文化內容策進院經費支持，承諾於合約期間或結案後，依文化內容策進院要求參與相關工作坊活動最多3場，分享提案內容與製作經驗交換。
  16. 本計畫案如獲文化內容策進院經費支持，承諾同意文化內容策進院基於推廣臺灣文化內容產業之相關行銷活動為前提之情況下，依該院要求協助行銷推廣相關活動。

文化內容策進院

個人資料蒐集、處理及利用同意書


立書人：（以下稱本人）

本人同意文化內容策進院（以下簡稱文策院）蒐集、處理及利用本人向文策院所提供的個人資料（包括但不限於以各項申請書表、法律文件、電子郵件等書面或非書面提供之個人資料，以下合稱個資），本人經文策院告知後同意以下事項：

  1. 本人同意文策院基於以下目的得蒐集、利用、處理個資：

    1. 文策院所提供之文化內容產業之相關服務或與提升文化內容之應用及產業化之目的範圍內利用。
    2. 文策院內部相關服務、內容之開發或改進，或作為內部稽核、資料分析及研究等使用。

  2. 個資蒐集類別

    本人同意文策院得蒐集本人個資類別，包括識別類（姓名、職稱、地址、聯絡電話、電子郵件信箱）、特徵類（年齡、性別、出生年月日等）、家庭情形類（配偶或同居人之姓名）、社會情況類（財產、執照或其他許可等）、教育、考選、技術或其他專業類（學歷等）、受僱情形類（現行受雇情形、工作經驗等）、財務資訊類(收入、所得、負債與支出、貸款等等)、商業資訊類(商業活動、與營業有關之執照等)、其他（書面文件之檢索等）。

  3. 個資利用之期間、地區、對象及方式

    1. 利用期間：個資蒐集之特定目的存續期間或依相關法令或契約約定執行業務所必須之保存期間。
    2. 利用地區：文策院個人資料主機、相關網路伺服機主機所在地及相關服務提供之地區。
    3. 利用對象：文策院、文策院業務委外單位及在文策院法定業務範圍內的合作往來單位。
    4. 利用方式：以人工、電腦化或自動化方式使用於本院各項服務與統計資料，或與前揭目的有關之利用。

  4. 本人知悉並確認以下事項：

    1. 本人擔保向文策院申請時均提供正確、最新及完整的個資。
    2. 若個資有任何異動，本人應立即主動向文策院更正，使其隨時保持正確。如因本人提供錯誤不實個資，或個資異動未及時向文策院更正，致本人權益受損，乃因本人行為所導致，與文策院無涉，文策院毋庸負責。
    3. 本人得就本人所提供之個資享有或行使以下權利：查詢或請求閱覽、請求製給複製本、請求補充或更正、請求停止蒐集、處理及利用、請求刪除之權利。本人如行使該權利致本人權益受損，與文策院無涉，文策院毋庸負責。
    4. 本人行使前項權利時，應以書面郵寄前開個人資料權利行使之文件至文策院登記地址行使上開權利。

  5. 準據法與管轄法院

    本同意書之解釋與適用均應依照中華民國法律處理，並以臺灣臺北地方法院為第一審管轄法院。
