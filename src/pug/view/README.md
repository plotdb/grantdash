# Customizable Views

Customizable views are pug / html files that should be able to be customized by org / brd. Following are all customizable views:

 * post-view.pug
 * prj-create.pug
 * prj-edit.pug
 * prj-list.pug
 * prj-view-simple.pug
 * prj-view.pug


## Routes and Rules

View files can be accessed via following ways:

 * nginx.config ( as static html )
 * rendered by express
